 FreelanceSaaS - Plateforme de Gestion Micro-SaaS

[![Vercel](https://img.shields.io/badge/Vercel-Deploy-black?style=flat&logo=vercel)](VOTRE_LIEN_VERCEL_ICI)
[![Next.js](https://img.shields.io/badge/Next.js-15-blue?style=flat&logo=next.js)](https://nextjs.org/)

Une application Full-Stack moderne conçue pour aider les freelances à gérer leurs clients, suivre leurs revenus en temps réel et exposer une API documentée.

🌟 Points Forts du Projet

- **Tableau de Bord Dynamique :** Visualisation des revenus mensuels avec Recharts.
- **API Publique Documentée :** Swagger UI intégré pour tester les points d'entrée `/api/clients`.
- **Base de Données Temps Réel :** Connecté à Supabase (PostgreSQL).
- **Sécurité :** Middleware de protection des routes et gestion des variables d'environnement.

🛠 Stack Technique

- **Frontend :** Next.js 15 (App Router), Tailwind CSS, Lucide React.
- **Backend :** Next.js API Routes, Zod (Validation).
- **Base de données :** Supabase (PostgreSQL).
- **Graphiques :** Recharts.

 Installation & Test

1. Cloner le dépôt : `git clone [URL_DU_REPO]`
2. Installer les dépendances : `npm install`
3. Créer un fichier `.env.local` avec vos clés Supabase.
4. Lancer en local : `npm run dev`


📖 Documentation API

Accédez à la documentation interactive sur `/api-doc` après avoir lancé l'application.
